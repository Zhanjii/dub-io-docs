## About

Dub.io is a desktop application that translates and dubs video content using ElevenLabs AI technology.

### Features

- **Video Dubbing** - Translate and dub videos to 29+ languages
- **Transcript Export** - Export subtitles in SRT, VTT, and TXT formats
- **Auto-Updates** - Automatic update checking and installation
- **Crash Reporting** - Optional error reporting for improved stability
