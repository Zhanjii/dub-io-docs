# Dub.io User Guide

## Getting Started

1. Launch the application
2. Enter your ElevenLabs API key in Settings
3. Select a video file to dub
4. Choose your target language
5. Click "Start Dubbing"

## Features

### Video Dubbing
Upload any video and translate the audio to your chosen language using AI.

### Subtitle Export
Export transcripts in multiple formats:
- SRT (SubRip)
- VTT (WebVTT)
- TXT (Plain text)

### Settings
- Configure your ElevenLabs API key
- Set default language preferences
- Enable/disable auto-updates

## Troubleshooting

### API Key Issues
Make sure your ElevenLabs API key is valid and has sufficient credits.

### Video Processing Errors
Ensure your video file is in a supported format (MP4, MOV, AVI, MKV).
