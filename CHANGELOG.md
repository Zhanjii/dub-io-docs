# Changelog

All notable changes to Dub.io will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-19

### Added
- Add admin PIN dialog component
- Add admin mode toggle, enhanced error handling, and expanded test coverage
- Add admin PIN manager and comprehensive test suite
- Add tool-finding and path utilities for bundled executables
- Add GitHub Pages documentation and enhanced About dialog
- Add auto-sync version to remote config and install on next startup
- Add outline width control and PIL-based subtitle preview
- Add subtitle styling options with live preview
- Add subtitle burn-in option for dubbed videos
- Add download quality options and fix SRT transcript export
- Add thumbnail preview, language autocomplete, and output path improvements
- Add auto-update system with Google Drive downloads
- Add changelog dialog and UI improvements
- Add encrypted activation code support
- Add automatic changelog and remote config system
- Add README.md with project overview

### Changed
- Add .DS_Store to gitignore
- Release: Version 1.0.0
- Initial commit: Dub.io project setup

### Improved
- Optimize app startup with lazy imports and deferred initialization

### Fixed
- Activation dialog centering and blank window issues
- Store user config in platform-specific locations
- Add macOS support for tool detection and window centering
- Show admin PIN in plain text instead of asterisks
- Add scrollable frame to dubbing tab to prevent content cutoff
- Use consistent tk widgets for thumbnail and subtitle preview
- Improve subtitle options with color picker and preview fix
- Update ElevenLabs SDK method calls for v2.30.0 compatibility
- Preserve app_access.json during changelog sync
